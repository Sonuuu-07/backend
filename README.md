# 🚀 API Launchpad Backend

This is a simple backend API built using Express.js.

## 📌 Features
- Get all users → /api/users
- Get user by ID → /api/users/:id
- API status check → /api/status

## 🛠️ Tech Stack
- Node.js
- Express.js

## ▶️ Run Locally

npm install  
npm start  

Server runs on:  
http://localhost:9000
